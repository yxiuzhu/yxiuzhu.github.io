### 发现定位精度问题
- 发现是因为滚动太快，计算的坐标位置还没有出来，因此使用sleep函数在页面滚动之后暂停0.3ms，顺便可以解决页面加载延迟的问题，这样就能够提升点击的定位精度
- 使用center函数，获取点击图片的中心位置，这样也能够提高定位精度
